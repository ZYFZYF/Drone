Blocking Send through the specified channel.

In both modes, using a localLocator of 0.0.0.0 will send through all whitelisted interfaces provided the channel is open.

Parameters
sendBufferSlice into the raw data to send.
sendBufferSizeSize of the raw data. It will be used as a bounds check for the previous argument. It must not exceed the sendBufferSize fed to this class during construction.
localLocatorLocator mapping to the channel we're sending from.
remoteLocatorLocator describing the remote destination we're sending to.
